Requirements:
- [Node v14](https://nodejs.org/download/release/latest-v14.x/)

Local Setup Steps:
1. ``git clone https://github.com/synassets/contracts.git ``
1. Install dependencies: `npm install`

1. Compile Solidity: ``npm run compile``
